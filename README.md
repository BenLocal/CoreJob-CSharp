# CoreJob-CSharp
基于[Quartz.Net](https://github.com/quartznet/quartznet "Quartz.Net")的任务调度系统，其中任务调度中心参照了[XXL-JOB](https://github.com/xuxueli/xxl-job "XXL-JOB")

调度中心访问地址：http://localhost:7800，默认登录账号 “admin/admin”。
